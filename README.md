# neko3cs dotfiles

## macos

run this.

```
$ sh -c "$(curl -fsSL https://raw.githubusercontent.com/neko3cs/dotfiles/master/macOS-bootstrap.sh)"
```

## ubuntu

run this.

```
$ sh -c "$(curl -fsSL https://raw.githubusercontent.com/neko3cs/dotfiles/master/ubuntu-bootstrap.sh)"
```
